# Animal Sounds Directory

This directory contains the audio files for the talking animals demo.

## Required Sound Files

The following MP3 files should be placed in this directory:

- `cow-moo.mp3` - Cow mooing sound
- `horse-neigh.mp3` - Horse neighing sound  
- `pig-oink.mp3` - Pig oinking sound
- `sheep-baa.mp3` - Sheep bleating sound
- `chicken-cluck.mp3` - Chicken clucking sound

## File Format

- **Format**: MP3
- **Quality**: 128kbps or higher recommended
- **Duration**: 2-5 seconds per sound
- **Sample Rate**: 44.1kHz

## Getting Sound Files

You can find free animal sound effects from:
- [Freesound.org](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- [SoundBible](http://soundbible.com/)

## Usage

These sounds are automatically loaded by the `HowlerAudioPlayer` component and the `Animal` components when users interact with the farm animals.

## Note

Make sure all sound files are properly licensed for your use case. The demo will work without these files, but users won't hear the animal sounds.
